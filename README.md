# Farcade Boy: The Run

An endless runner built for **[Remix](https://remix.gg)** — sprint through a
stylized world, dodge hazards, grab stars and chase a high score.

🎮 **[Play it on Remix »](https://remix.gg/g/8snnRpYVY5pz)** · single-file HTML5 game

## About
Fast-paced endless runner with shield & star power-ups, a day–night sky cycle and
layered parallax backgrounds. Mobile-first, portrait, one-touch controls.
